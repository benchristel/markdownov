# Technology Stack

## Core Technologies
- **Programming Language:** TypeScript
- **Runtime Environment:** Node.js (v23.11.0) & Bun (v1.2.13)
- **Package Manager:** pnpm (v9.10.0)

## Development & Tooling
- **Testing Framework:** Taste (`@benchristel/taste`)
- **Linguistic Analysis:** `pos` (Part-of-speech tagger)
- **Build/Task Automation:** Makefile
- **Linting/Styling:** ESLint with Stylistic plugin
- **Version Control:** Git with Husky hooks
- **Release Management:** `bumpp`
